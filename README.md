# CLI-GHA-DICTIONARY

greek-hebrew-aramaic dictionary

CLI app to consult the Strongs dictionary (more dictionaries in the future).

first, runt the build.sh and the 'gha' is the app compiled


## EXAMPLES

search strongs entries in greek

  > gha -g 2
  
result:

>G00002  Ἀαρών   Aarṓn   ah-ar-ohn'      2 of Hebrew origin ( ); Aaron, the brother of Moses :--Aaron.

get strongs entries  in hebrew

  > gha -h 2

result:

>H2      אַב      אב      ʼab     [G:references: G5,G912] 1) father (Aramaic)corresponding to {father} father.


reindex the idx files.

  > gha -r
